Inkscape is a professional quality vector graphics software with sophisticated drawing and manipulation tools. It has a comprehensive feature set that is used by professionals and hobbyists around the world.

Notable Features

Drawing tools

At the heart of Inkscape are sophisticated drawing tools that allow freehand drawing or bezier curves.

Node points

Inkscape's node tool enables precision editing of multiple paths at once.

Text tool

Inkscape has on-canvas text editing with precise control over typography.

XML editor

Directly manipulate underlying SVG code to ensure complete W3C compliancy.

Creation

Shape tools

Give quick expression to your creativity with ready to use rectangles, arcs, stars and polygons.

3D boxes

Create cubes controlling vanishing points.

Gradients

Inkscape allows easy onscreen editing of complex gradients.

Clones

Inkscape clones are "live" linked copies of objects that change with the original, but can also be transformed independently.

Bitmaps

Import raster graphics by either linking or embedding them, and perform basic editing. You can also create bitmaps of selected objects.

Connectors

Creating organization charts or flow diagrams is easy with Inkscape's connector tool.

Manipulation

Transformations

Inkscape supports a number of object transformations, such as scale, rotate and skew.

Tweak tool

Alter objects shape and style with a sweep of your mouse.

Path effects

Non-destructive effects helps you manipulate your objects in creative ways.

Styling

Inkscape has intuitive fill and stroke dialogs that control fill and stroke colour, and stroke styles. The colour palette bar allows quick changes.

Clips & masks

Selectively hide parts of your objects without having to cut them.

Layers

Organise elements into a logical structure. You can lock, hide or adjust opacity and blending mode of individual layers.

Positioning

Positioning objects in Inkscape is simple with the selector tool, the align & distribute dialog, and z-order operations.

Object grouping

Inkscape lets you group related objects for easier manipulation.

More stuff

Spiro splines

With this kind of curves creating smooth round paths is a piece of cake.

Grids

Create as many rectangular and axonometric grids as you need. Versatile settings lets you fine tune colour, spacing, angle and more.

Guides

Create ortogonal and angled guides or move and rotate any guide on canvas.

Snapping

Powerful snapping of objects, paths and gradients. Snap everything to everything with the finest control and contextual visual aids.

Swatches (Auto palette)

Create colours and reuse them everywhere for easier styling of your objects.

Filters

Enhance your art with a wide selection of visual effects, all SVG compliant.

Extensions

Python-powered scripts lets you go beyond base editing tools capabilities.

Color management

Inkscape uses industry renown littleCMS for calibrated colours, display profiling and basic print simulation.

Excelent interoperability

Open and save to multiple graphic formats. UniConvertor provides compatibility for some of these formats.

Free!

Inkscape cost no money and it's open source software released under the GPL2 license.
